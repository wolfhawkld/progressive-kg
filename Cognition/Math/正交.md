# 正交

> 两向量内积为零，方向独立互不影响；深度学习中用于初始化和优化。

---

## 向量正交

两个向量 u、v 正交，当且仅当 u·v=0。几何意义是夹角为 90°，一个向量在另一个上的投影为零。

- **独立性**：正交向量互不包含对方的分量
- **最远距离**：同长度向量中，正交时距离最大
- **投影为零**：u 在 v 上的投影长度为 0

### 定义推导

正交的定义依赖 [[内积]]。两个向量 u、v 正交当且仅当：

```
u · v = Σ(u_i × v_i) = 0
```

几何意义：夹角 θ 满足 cos(θ) = 0，即 θ = 90°。

### 正交向量组

一组向量 {v₁, v₂, ..., vₙ} 两两正交：v_i · v_j = 0 (i ≠ j)。若还满足 ||v_i|| = 1，则称为正交归一化（Orthonormal）。

---

## 正交矩阵

矩阵 Q 是正交矩阵，当 Q^T Q = I。等价条件：列向量正交归一化，Q^T = Q^{-1}。

- **保长度**：||Qx|| = ||x||
- **保角度**：(Qx)·(Qy) = x·y
- **逆矩阵**：Q^{-1} = Q^T（计算极快）
- **行列式**：det(Q) = ±1

### 性质证明

由 Q^T Q = I：

```
||Qx||² = (Qx)·(Qx) = x^T Q^T Q x = x^T I x = x^T x = ||x||²
```

因此 ||Qx|| = ||x||，即正交变换保长度。

### 常见应用

- **旋转矩阵**：2D/3D 旋转
- **坐标变换**：FFT、小波变换
- **QR 分解**：A = QR，Q 是正交矩阵

---

## 正交初始化

将权重矩阵初始化为正交矩阵，防止梯度消失或爆炸。

- **防梯度问题**：正交矩阵保长度，信号强度不随层衰减或放大
- **加速收敛**：信息流更稳定，初期收敛更快
- **PyTorch 一行**：`torch.nn.init.orthogonal_(W)`

### PyTorch 代码

```python
import torch
W = torch.empty(m, n)
torch.nn.init.orthogonal_(W)
```

### 与 [[标准化]] 的关系

正交初始化作用于权重矩阵（保持信号幅度），标准化作用于激活值（控制分布）。两者互补：正交初始化稳定前向传播，标准化稳定反向传播。

## Muon 优化器中的正交化

Muon 使用 Newton-Schulz 迭代对动量矩阵进行正交化，提取正交因子 U V^T（SVD 极分解）。

- **msign(M)** = M(M^T M)^{-1/2}
- **效果**：保留方向信息，抑制幅度偏差
- **迭代**：X_{k+1} = X_k(3 - X_k^T X_k) / 2，收敛到正交矩阵

### Newton-Schulz 迭代

```
输入：矩阵 M
输出：正交化矩阵 Q ≈ msign(M)

迭代：
  X_0 = M / ||M||_F
  X_{k+1} = X_k * (3I - X_k^T X_k) / 2
```

通常 3-5 次迭代即可收敛。

## 正交化方法

### Gram-Schmidt 正交化

将一组向量转化为正交向量组：

```
u₁ = v₁
u₂ = v₂ - (v₂·u₁)/(u₁·u₁) × u₁
u₃ = v₃ - (v₃·u₁)/(u₁·u₁) × u₁ - (v₃·u₂)/(u₂·u₂) × u₂
```

### QR 分解

任何矩阵 A 可分解为 A = QR，其中 Q 是正交矩阵，R 是上三角矩阵。
