# 均方误差（Mean Squared Error, MSE）

> 衡量预测值与真实值之间偏差程度的常用指标，核心是计算差的平方的平均值。

---

## 核心定义
对于包含 n 个样本的数据集，真实值为 y_i，预测值为 ŷ_i：

```
MSE = (1/n) Σ (y_i - ŷ_i)²
```

**本质**：误差平方的期望值。

---

---

核心特点

| 特点 | 说明 |
|------|------|
| **忽略方向** | 只考虑误差大小，不考虑正负 |
| **惩罚大误差** | 平方使偏离大的预测受到更重惩罚 |
| **数学特性好** | 连续可微，便于梯度计算 |
| **单位问题** | 结果单位是原单位的平方 |

---

## 损失函数视角

MSE 常被称为**平方损失**，也常口语化地归入 L2 损失。严格说，L2 范数是平方和再开方，而训练中常用的是平方 L2 范数或其均值。

**优势**：
- 对预测值 $\hat y$ 是凸函数；在线性模型参数上也是凸的，但唯一解还需要设计矩阵满秩或额外约束
- 梯度计算简单
- 在独立同方差高斯噪声假设下对应最大似然估计

**劣势**：
- 对异常值过于敏感（可能过拟合噪声）
- 不是鲁棒损失
- 对深度网络参数通常是非凸目标，不能由 MSE 本身保证全局最优

---

## 与相关指标对比

| 指标 | 公式 | 特点 |
|------|------|------|
| **MSE** | (1/n)Σ(y-ŷ)² | 平方惩罚，单位为原单位² |
| **RMSE** | √MSE | 平方根还原单位，更直观 |
| **MAE** | (1/n)Σ|y-ŷ| | 绝对值，对异常值更鲁棒 |
| **MAPE** | (100/n)Σ|y-ŷ|/|y| | 无量纲，但真实值为 0 或接近 0 时不稳定，负值也难解释 |

---

## Python 实现

```python
import numpy as np
from sklearn.metrics import mean_squared_error

# 手动计算
y_true = np.array([1, 2, 3, 4, 5])
y_pred = np.array([1.1, 1.9, 3.2, 3.8, 5.1])
mse = np.mean((y_true - y_pred) ** 2)

# sklearn
mse = mean_squared_error(y_true, y_pred)
```

## 与其他损失函数的选择

| 场景 | 推荐 |
|------|------|
| 误差成本近似二次、噪声接近高斯 | MSE |
| 存在异常值、需鲁棒性 | MAE 或 Huber Loss |
| 预测比例/百分比 | 先确认无零值、符号与业务权重，再考虑 MAPE 或替代指标 |
| 分类问题 | Cross-Entropy（不是 MSE） |

## 数学性质

MSE 的优化性质与统计解释属于不同层面：前者研究当前预测的导数，后者还要对训练集和噪声取期望。

### 可微性

```
∂MSE/∂ŷ_i = (2/n)(ŷ_i - y_i)
```

梯度线性，便于优化。

### 偏差—方差分解

对固定输入 $x$，把训练数据集的随机性也纳入期望，并假设 $y=f(x)+\varepsilon$、$E[\varepsilon]=0$，有：

$$
E[(y-\hat f(x))^2]
=\operatorname{Bias}[\hat f(x)]^2
+\operatorname{Var}[\hat f(x)]
+\operatorname{Var}(\varepsilon)
$$

最后一项是不可约噪声。它不是把某一批残差直接写成“误差方差 + 误差偏差平方”，分解对象和期望范围必须说明。
