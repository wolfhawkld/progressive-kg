# 均方误差（Mean Squared Error, MSE）

> 衡量预测值与真实值之间偏差程度的常用指标，核心是计算**差的平方的平均值**。

---

## 定义

对于包含 n 个样本的数据集，真实值为 y_i，预测值为 ŷ_i：

```
MSE = (1/n) Σ (y_i - ŷ_i)²
```

**本质**：误差平方的期望值。

---

## 核心特点

| 特点 | 说明 |
|------|------|
| **忽略方向** | 只考虑误差大小，不考虑正负 |
| **惩罚大误差** | 平方使偏离大的预测受到更重惩罚 |
| **数学特性好** | 连续可微，便于梯度计算 |
| **单位问题** | 结果单位是原单位的平方 |

---

## 损失函数视角

MSE 又被称为 **L2 损失** 或 **平方损失**，是回归问题最常用的损失函数。

**优势**：
- 凸函数，全局最优解唯一
- 梯度计算简单
- 对异常值敏感（可迫使模型拟合）

**劣势**：
- 对异常值过于敏感（可能过拟合噪声）
- 不是鲁棒损失

---

## 与相关指标对比

| 指标 | 公式 | 特点 |
|------|------|------|
| **MSE** | (1/n)Σ(y-ŷ)² | 平方惩罚，单位为原单位² |
| **RMSE** | √MSE | 平方根还原单位，更直观 |
| **MAE** | (1/n)Σ|y-ŷ| | 绝对值，对异常值更鲁棒 |
| **MAPE** | (100/n)Σ|y-ŷ|/y | 百分比，适合比例数据 |

---

## Python 实现

```python
import numpy as np
from sklearn.metrics import mean_squared_error

# 手动计算
y_true = np.array([1, 2, 3, 4, 5])
y_pred = np.array([1.1, 1.9, 3.2, 3.8, 5.1])
mse = np.mean((y_true - y_pred) ** 2)

# sklearn
mse = mean_squared_error(y_true, y_pred)
```

## 应用场景

| 领域 | 应用 |
|------|------|
| **回归模型评估** | 线性回归、神经网络回归的评价指标 |
| **损失函数** | MSE Loss 用于训练回归模型 |
| **参数估计** | 数理统计中参数估计的精度度量 |
| **信号处理** | 降噪、压缩的质量评估 |

---

## 与其他损失函数的选择

| 场景 | 推荐 |
|------|------|
| 数据干净、无异常值 | MSE ✅ |
| 存在异常值、需鲁棒性 | MAE 或 Huber Loss |
| 预测比例/百分比 | MAPE |
| 分类问题 | Cross-Entropy（不是 MSE） |

---

## 数学性质

### 可微性

```
∂MSE/∂ŷ_i = (2/n)(ŷ_i - y_i)
```

梯度线性，便于优化。

### 统计意义

```
MSE = Var(误差) + [Bias(误差)]²
```

MSE = 方差 + 偏差平方（偏差-方差分解）
