# 幂变换

> 对数据施加幂函数变换以改变分布形态，校正偏态、稳定方差、逼近正态分布。

---

## 核心思想

幂变换的核心是给数据加一个 λ 次幂，通过调节 λ 改变分布的形状。当数据右偏（长尾在右）时，用 λ < 1 的幂变换压缩大值、拉伸小值，使分布更对称。

- **偏态校正**：右偏数据 → 用 λ < 1（如对数 √ 倒数），左偏数据 → 用 λ > 1（如平方立方）
- **方差稳定**：幂变换可以消除"均值越大方差越大"的异方差性（heteroscedasticity）
- **正态逼近**：找到最佳 λ 使变换后的数据最接近正态分布，满足线性模型的假设

### 一般形式

```
x' = x^λ
```

- λ = 1：不变换
- λ = 0.5：平方根变换
- λ = 0：对数变换（log x，实际是 Box-Cox 在 λ→0 时的极限）
- λ = -1：倒数变换（1/x）
- λ = -0.5：倒数平方根

### 偏态与 λ 的选择

右偏（long right tail）的数据，如收入、房价、点击量：
- 对数变换（λ=0）是最常用的选择
- 压缩大数值的间距，拉伸小数值的间距
- 使极端值不再是"孤岛"

---

## Box-Cox 变换

最广泛使用的参数化幂变换，通过极大似然估计自动选择最优 λ。要求输入数据必须全部为正。

- **公式**：x' = (x^λ - 1) / λ（λ≠0），x' = log(x)（λ=0）
- **λ 选择**：通过 Box-Cox 归一化后的对数似然函数自动估计
- **限制**：x > 0（严格正数），有零或负值的数据不可用

### 公式细节

Box-Cox 的分段定义保证 λ 连续（在 λ=0 处是 log 变换的极限）：

```
x'(λ) = { (x^λ - 1) / λ,  λ ≠ 0
       { log x,             λ = 0
```

- 为什么要减 1 除以 λ？保证变换在 λ 连续——当 λ→0 时 (x^λ - 1)/λ → log x
- λ 通常通过极大似然估计或 profile likelihood 找到最优值
- scikit-learn 的 `PowerTransformer(method='box-cox')` 自动完成此过程

### 应用场景

| 场景 | 原因 | 典型 λ |
|---|---|---|
| 线性回归 | 满足正态性假设 | 自动估计 |
| ANOVA | 满足方差齐性 | 自动估计 |
| 时间序列 | 稳定方差，改善预测 | 通常 λ=0（对数） |
| 经济数据（GDP、收入） | 右偏严重 | 0~0.3 |

## Yeo-Johnson 变换

Box-Cox 的扩展，支持零和负值。不需要数据严格为正，适用性更广。

- **公式**：分段处理正负值，保证变换连续可微
- **正数**：与 Box-Cox 类似
- **零和负数**：通过平移处理
- **自动 λ 选择**：同 Box-Cox 的极大似然估计

### 何时用 Yeo-Johnson 代替 Box-Cox

| 条件 | 推荐方法 | 理由 |
|---|---|---|
| 数据全为正 | Box-Cox 或 Yeo-Johnson | Box-Cox 更常见，论文更易理解 |
| 数据含零或负值 | Yeo-Johnson | Box-Cox 不支持非正数 |
| 标准化流程（如模型 pipeline） | Yeo-Johnson | 一次适配，任何数据都能处理 |

scikit-learn 的 `PowerTransformer(method='yeo-johnson')` 是默认方法。

## 与标准化的关系

幂变换和标准化是数据预处理的两个步骤，幂变换在前、标准化在后，解决不同层次的问题。

- **幂变换目标**：改变分布形态（校正偏态、稳定方差），一般不改变居中趋势
- **标准化目标**：消除量纲差异（减均值除标准差），不改变分布形状
- **使用顺序**：先做幂变换校正分布，再做 Z-score 标准化消除量纲
- **类比**：幂变换是"把歪的砖石雕正"，标准化是"把所有砖石放到统一刻度尺上"

### 典型的双步骤流程

```python
from sklearn.preprocessing import PowerTransformer, StandardScaler

y_xform = PowerTransformer(method='yeo-johnson').fit_transform(X)
y_scaled = StandardScaler().fit_transform(y_xform)
```

### 什么时候可以跳过幂变换

- 数据已经近似正态分布
- 模型对分布形态不敏感（树模型如 XGBoost、LightGBM）
- 特征本身就是对称的（如许多图像像素值或标准化后的特征）

[[标准化]] 是几乎所有基于距离/梯度的模型的标配；幂变换是仅在数据偏态严重时的可选前置步骤。
