# 自指（Self-reference）

> 一个系统**指向自身**、**谈论自身**或**包含自身**的现象。是《哥德尔、艾舍尔、巴赫》的核心主题。

---

> 自指是指某个对象、陈述或系统在其内容中**引用自身**，形成一种特殊的循环结构。

```
"这句话是假的。" — 自指悖论的经典例子
```

---

---

核心价值

自指揭示了一个深刻问题：**形式系统的边界**

- 完备的系统无法包含自指（会导致悖论）
- 自指的系统无法完备（哥德尔定理）

这是理解**意识、智能、创造性**的关键线索。

---

## GEB 三位一体

侯世道在《GEB》中展示了自指在三个领域的表现：

| 领域 | 代表人物 | 自指形式 | 例子 |
|------|----------|----------|------|
| **数学** | 哥德尔 | 形式系统的自我否定 | 哥德尔不完备定理 |
| **艺术** | 艾舍尔 | 视觉循环 | 《画手》、《瀑布》 |
| **音乐** | 巴赫 | 结构递归 | 《赋格》、《卡农》 |

### 哥德尔不完备定理

```
在任何足够强大的形式系统中，
存在一个命题 G，使得：
- G 说："G 不可证明"
- 如果 G 可证明 → G 是假的 → 系统不一致
- 如果 G 不可证明 → G 是真的 → 系统不完备
```

**结论**：完备性与一致性**不可兼得**。

### 艾舍尔的画作

- **《画手》**：两只手互相画对方
- **《瀑布》**：水流沿"向上"的路径循环
- **《画廊》**：画中的人在看他所在的画廊

视觉上的不可能 → 逻辑上的悖论 → 形式系统的边界

### 巴赫的音乐

- **卡农**：旋律与自身的变奏叠加
- **赋格**：主题在不同声部递归出现
- **不断升高的卡农**：听觉上的"无限上升"

## 计算机科学中的自指

### 递归

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)  # 函数调用自身
```

递归 = 程序的自指

### 自修改程序

程序可以修改自身的代码（如 Lisp 的 `eval`）

### Quine（自复制程序）

```python
s = 's = %r; print(s %% s)'
print(s % s)
```

程序打印出**自己的源代码**。

## 与意识的关联

侯世道的核心命题：

> **意识 = 自指系统**

- 大脑思考大脑本身
- "我"的概念是对自我的引用
- 元认知 = 关于认知的认知

自指可能是**意识涌现**的机制。

---

## 怪圈（Strange Loop）

自指形成的层级交叉结构：

```
层级 A → 层级 B → 层级 C → ... → 回到层级 A
```

- 看起来在上升，实际回到起点
- 悖论、不可能、涌现都来自怪圈
