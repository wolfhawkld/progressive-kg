# 自指（Self-reference）

> 表达式、对象或系统通过名称、编码或层级关系指向自身的结构。

## 核心定义

自指可以是直接的，例如一句话谈论“这句话”；也可以通过编码和替换间接构造。自指本身既不必产生悖论，也不自动产生智能或意识。

| 结构 | 是否等同自指 | 说明 |
|---|---|---|
| 递归 | 不等同 | 规则引用同一规则的较小实例，并可由基例终止 |
| 循环依赖 | 不等同 | A 依赖 B、B 依赖 A，不一定有表示层面的“谈论自身” |
| 固定点 | 相关但不同 | $f(x)=x$ 描述不动点，不要求表达式引用自己 |
| 反射/元编程 | 可用于自指 | 程序读取或操作自身表示，但普通 `eval` 不等于修改自身代码 |

“这句话是假的”展示语义真值中的说谎者悖论；它不是哥德尔不完备定理的完整证明。

## 哥德尔不完备定理的边界

对一致、可有效公理化且足以表达一定初等算术的形式理论，第一不完备定理说明存在该理论既不能证明也不能否证的句子。第二不完备定理在相应条件下说明该理论不能证明表达其自身一致性的标准句子。

证明使用算术化和对角引理构造具有间接自指性质的句子，但不能简化为：

- “任何包含自指的系统都不完备”；
- “完备系统无法表达自指”；
- “一致性与完备性对所有逻辑系统都不可兼得”。

较弱、有限或使用不同语义/逻辑的系统可能同时一致且完备。定理讨论的是特定形式理论中的可证明性，不直接证明人类心智或 AI 的能力上限。

## 程序中的自指

程序可通过输出、解释或操作自身表示形成自指；递归只是其中相关但不同的一种结构。

### Quine

Quine 输出自身源代码，不读取源文件：

```python
s = 's = %r; print(s %% s)'
print(s % s)
```

它依赖“程序表示”和“执行结果”两个层级之间的构造。

### 递归与自解释器

递归函数通常只是调用同一规则；自解释器则用某语言实现该语言的解释过程。二者都可形成层级回环，但概念重点不同。

## 艺术、音乐与怪圈

Hofstadter 用哥德尔句、Escher 的视觉回环和 Bach 的卡农/赋格讨论“strange loop”：系统在层级间移动后回到自身。这个框架富有解释力，但它把形式逻辑、艺术结构和认知类比并置，不表示三者共享同一个数学定理。

- 逻辑部分讨论形式系统中的编码与可证明性。
- 视觉和音乐部分展示感知上的层级回环。
- “怪圈”是连接这些例子的解释框架，而非统一证明。

## 与意识的关系

“自指或怪圈构成自我意识”是 Hofstadter 等人的哲学与认知理论主张，不是已确证的神经科学结论。评估此类主张需要区分：

- 系统能否表示自己的状态；
- 能否监控并修正行为；
- 是否具有主观体验；
- 观察到的元认知行为能否由非自指机制解释。

因此本库可以把自模型、元认知和自我改进作为研究假设连接起来，但不应写成“意识 = 自指系统”的事实等式。
