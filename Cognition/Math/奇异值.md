# 奇异值（Singular Value）

> 奇异值是矩阵在不同正交方向上的“拉伸倍率”。它告诉我们：一个线性变换会把空间沿哪些方向拉长/压扁，以及每个方向拉伸多少。

---

## 直觉理解

把矩阵 A 看成一个“空间变形器”：

```text
输入空间中的单位圆/单位球
        ↓  经过矩阵 A 变换
输出空间中的椭圆/椭球
```

奇异值就是这个椭圆/椭球各条主轴的长度。

- 最大奇异值 σ₁：矩阵能把某个方向拉伸到多大
- 最小奇异值 σ_min：矩阵对最弱方向还保留多少信息
- 接近 0 的奇异值：说明某些方向被压扁，信息可能丢失

一句话：

> 特征值看“矩阵对自身空间的固有方向怎么缩放”，奇异值看“矩阵作为变换整体对空间怎么拉伸”。

---

## 核心定义/公式

对任意实矩阵 A ∈ R^{m×n}，它的奇异值来自 A^T A 的特征值：

```text
A^T A v_i = λ_i v_i
σ_i = sqrt(λ_i)
```

其中：

- λ_i 是 A^T A 的非负特征值
- σ_i ≥ 0 是 A 的奇异值
- 通常按从大到小排序：σ₁ ≥ σ₂ ≥ ... ≥ σ_r > 0

奇异值分解（SVD）写作：

```text
A = U Σ V^T
```

含义：

| 符号 | 含义 |
|------|------|
| V^T | 先把输入旋转/反射到一组正交方向 |
| Σ | 沿各个正交方向按奇异值缩放 |
| U | 再旋转/反射到输出空间 |

所以矩阵 A 可以理解为：

```text
旋转 → 拉伸/压缩 → 旋转
```

---

## 与特征值的区别

| 概念 | 适用对象 | 数值可能性 | 几何含义 |
|------|----------|------------|----------|
| 特征值 Eigenvalue | 通常要求方阵 | 可正、负、复数 | 固有方向上的缩放/翻转/旋转 |
| 奇异值 Singular Value | 任意矩阵 | 一定非负实数 | 正交主方向上的拉伸倍率 |

重要关系：

```text
奇异值(A) = sqrt(特征值(A^T A))
```

如果 A 是实对称且半正定矩阵，那么奇异值与特征值相同；但一般情况下二者不同。

---

## 关键性质/特点

| 性质 | 说明 |
|------|------|
| 非负性 | 奇异值总是 ≥ 0 |
| 任意矩阵可用 | 不要求方阵、不要求可对角化 |
| 衡量拉伸 | σ_i 表示第 i 个正交方向上的缩放倍率 |
| 衡量秩 | 非零奇异值个数 = rank(A) |
| 衡量条件数 | cond(A)=σ_max/σ_min，越大越病态 |
| 最优低秩近似 | 保留最大 k 个奇异值可得到最佳 rank-k 近似 |

## 几何图像

二维情况下：

```text
单位圆
  ↓ A
椭圆
```

- 椭圆长轴长度 = 最大奇异值 σ₁
- 椭圆短轴长度 = 第二奇异值 σ₂
- 长轴方向 = 左奇异向量 u₁
- 输入中对应方向 = 右奇异向量 v₁

如果某个奇异值为 0，说明该方向被压缩成 0，矩阵不可逆或信息丢失。

## 深度学习中的应用

### 1. 梯度稳定性

神经网络中，线性层的权重矩阵会不断放大或缩小激活与梯度。

- σ_max 很大：可能导致梯度爆炸
- σ_min 很小：可能导致梯度消失
- 奇异值集中在 1 附近：信息传递更稳定

这也是 [[正交]] 初始化和正交约束有用的原因：正交矩阵的奇异值全为 1。

### 2. 谱范数与 Lipschitz 控制

矩阵的谱范数等于最大奇异值：

```text
||A||₂ = σ_max(A)
```

在深度学习中可用于：

- 谱归一化（Spectral Normalization）
- 控制模型 Lipschitz 常数
- 提升 GAN、判别器、鲁棒模型稳定性

### 3. 低秩压缩

SVD 可保留最大的 k 个奇异值：

```text
A ≈ U_k Σ_k V_k^T
```

应用：

- 模型压缩
- LoRA/低秩适配的直觉基础
- 权重矩阵结构分析
- embedding / hidden state 的主方向分析

### 4. Muon 优化器中的矩阵正交化

[[Muon优化器]] 需要把动量矩阵近似投影到“更像正交更新”的方向。精确做法可以通过 SVD 得到极分解中的正交因子：

```text
A = U Σ V^T
orthogonal_factor = U V^T
```

但直接 SVD 成本较高，所以 Muon 使用 Newton-Schulz 迭代近似完成类似效果。

## Python 实现（简化版）

```python
import numpy as np

A = np.array([
    [3.0, 2.0],
    [0.0, 1.0],
    [0.0, 0.0],
])

# SVD: A = U @ diag(S) @ Vt
U, S, Vt = np.linalg.svd(A, full_matrices=False)

print("singular values:", S)
print("rank:", np.sum(S > 1e-8))
print("spectral norm:", S[0])
print("condition number:", S[0] / S[-1])
```

PyTorch：

```python
import torch

A = torch.randn(4, 3)
S = torch.linalg.svdvals(A)

sigma_max = S[0]
rank = torch.sum(S > 1e-6)
```
