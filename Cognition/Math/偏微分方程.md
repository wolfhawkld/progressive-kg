# 偏微分方程

> 含多元函数偏导数的方程，描述各变量变化率间的约束关系，物理建模的核心工具。

---

## 基本概念

偏微分方程（PDE）是包含未知多元函数及其偏导数的方程。与常微分方程（ODE）的区别在于：ODE 只涉及一个自变量的导数，PDE 涉及多个自变量的偏导数。

- **偏导数**：对某一变量求导时，其他变量视为常数
- **阶**：方程中最高阶偏导数的阶数
- **线性/非线性**：未知函数及其导数是否以线性形式出现
- **齐次/非齐次**：方程是否有不含未知函数的项（源项）

### 偏导数定义

函数 u(x, y) 对 x 的偏导数：

```
∂u/∂x = lim[h→0] (u(x+h, y) - u(x, y)) / h
```

几何意义：在 y 固定的截面上，u 沿 x 方向的变化率。

### 阶与线性性

二阶线性 PDE 的一般形式：

```
A ∂²u/∂x² + B ∂²u/∂x∂y + C ∂²u/∂y² + D ∂u/∂x + E ∂u/∂y + F u = G
```

- 线性：A~G 只是 x, y 的函数，不依赖 u
- 非线性例子：Navier-Stokes 方程（对流项 u·∇u 是非线性的）

---

## 经典方程

三类二阶线性 PDE 对应三种物理过程，是 PDE 理论的核心。

- **波动方程**：振动传播（弦、声波、电磁波）
- **热传导方程**：扩散过程（热、浓度、概率）
- **拉普拉斯方程**：稳态平衡（电势、流体、温度平衡）

### 波动方程

```
∂²u/∂t² = c² ∇²u
```

- 描述：弦振动、声波、光波
- 类型：双曲型
- 解的特性：有限传播速度 c，波前清晰

### 热传导方程

```
∂u/∂t = α ∇²u
```

- 描述：热扩散、粒子布朗运动、概率分布演化
- 类型：抛物型
- 解的特性：无限传播速度（理论上），信号瞬间到达所有点但指数衰减

### 拉普拉斯方程

```
∇²u = 0
```

- 描述：静电场、稳态温度、不可压缩无旋流
- 类型：椭圆型
- 解的特性：解在其定义域内无限可微（调和函数的平滑性）

---

## 分类与特征

二阶 PDE 按判别式 B² - 4AC 分类，不同类型有截然不同的解行为。

- **椭圆型**（B² - 4AC < 0）：拉普拉斯方程——稳态，全局耦合
- **抛物型**（B² - 4AC = 0）：热传导方程——演化，平滑扩散
- **双曲型**（B² - 4AC > 0）：波动方程——传播，有限速度

### 特征线理论

双曲型方程的信息沿特征线传播。特征线的斜率由方程系数决定：

```
dy/dx = (B ± √(B² - 4AC)) / (2A)
```

- 解在特征线上满足特定的相容关系
- 跨越特征线时解可能不连续（激波）
- 椭圆型方程无实特征线——信息全局耦合，无传播方向

### 适定性（Well-posedness）

Hadamard 适定性三条件：
1. 解存在
2. 解唯一
3. 解连续依赖初始/边界条件

不同类型的 PDE 需要不同的定解条件：
- 椭圆型：边界条件（Dirichlet/Neumann）
- 抛物型：初始条件 + 边界条件
- 双曲型：两个初始条件（位置 + 速度）+ 边界条件

## 求解方法

PDE 求解方法分为解析法和数值法，实际工程以数值法为主。

- **分离变量法**：利用[[正交]]函数族展开解
- **傅里叶变换**：将 PDE 转为 ODE
- **有限差分法**：离散网格上近似偏导数
- **有限元法**：变分原理 + 分片逼近

### 分离变量法

假设解可分离 u(x, t) = X(x)T(t)，代入 PDE 后分离变量：

```
波动方程：X''T = c² XT''  →  X''/X = T''/(c²T) = -λ
```

得到两个 ODE，解出 X(x) 和 T(t)，利用边界条件确定 λ。X(x) 的解构成正交函数族（如 sin(nπx/L)），通解是正交展开：

```
u(x, t) = Σ [aₙ cos(nπct/L) + bₙ sin(nπct/L)] sin(nπx/L)
```

这正是[[正交]]性在 PDE 中的核心应用——Sturm-Liouville 理论保证特征函数正交完备。

### 傅里叶变换法

对空间变量做傅里叶变换，PDE 变为时间的 ODE：

```
∂u/∂t = α ∇²u  →  ∂û/∂t = -α|k|² û  →  û(k, t) = û(k, 0) e^{-α|k|²t}
```

高频分量衰减更快——这就是扩散的本质：细节先消失，粗结构后消失。

### 有限差分法

网格 (i, n) 上用差商近似偏导数：

```
∂u/∂t ≈ (u_i^{n+1} - u_i^n) / Δt     (向前差分)
∂²u/∂x² ≈ (u_{i+1} - 2u_i + u_{i-1}) / Δx²   (中心差分)
```

稳定性条件（CFL 条件）：Δt / Δx² ≤ 1/(2α)（热传导），cΔt/Δx ≤ 1（波动）。

## 与深度学习的关联

PDE 在机器学习中既可以描述概率密度的演化，也可以作为神经网络要逼近的物理规律。需要区分：单个状态随时间演化通常由 ODE/SDE 描述，状态的空间分布或空间场才通常由 PDE 描述。

- **扩散模型**：样本路径由 SDE 描述，对应概率密度由 Fokker–Planck PDE 描述
- **神经 ODE**：残差网络连续化后得到 ODE；只有引入空间变量时才自然成为 PDE
- **梯度流**：参数梯度流是 ODE，某些“概率分布上的梯度流”才与扩散型 PDE 等价

### 扩散模型

[[扩散模型]] 的前向过程用 SDE 逐步扰动数据分布。例如常见的 variance-preserving SDE：

```
dx = -½β(t)x dt + √β(t) dW
```

- 前向 SDE 诱导概率密度 $p_t(x)$ 按 Fokker–Planck PDE 演化
- 逆时间 SDE 的漂移项依赖 score $\nabla_x \log p_t(x)$，通常由神经网络估计
- 只有特定情形退化为经典热方程；不能把所有扩散过程都等同于热传导

### 神经 ODE

残差网络 x_{l+1} = x_l + f(x_l) 在连续极限下变为 ODE：

```
dx/dt = f(x, t)
```

- 层数 → 连续时间 t
- 伴随状态法通过反向积分另一个 ODE 计算梯度，可降低保存全部中间状态的需求
- 数值误差和求解成本会影响是否采用纯伴随法、检查点或直接反向传播
- 当状态本身是空间场、动力学含空间微分算子时，模型才进入神经 PDE 范畴

### 梯度流视角

连续时间极限下，[[梯度下降优化]] 对参数的演化是梯度流 ODE：

```
dθ/dt = -∇L(θ)
```

- 这不是参数空间中的热方程，学习率也不能直接解释为热扩散系数
- 随机梯度噪声可在近似条件下写成 SDE，但噪声结构通常不是各向同性常数扩散
- 在 Wasserstein 空间中，概率分布对熵泛函的梯度流可得到热方程；这是一个特定的分布空间结论

## 关系网络

- 对比 [[梯度下降优化]] — 参数梯度流通常是 ODE，而非热传导 PDE
