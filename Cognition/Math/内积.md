# 内积（Inner Product）

> 两个向量之间的"相似度度量"运算，结果是**一个数值**。

---

## 定义

对于两个向量 **a** 和 **b**：

```
a · b = |a| × |b| × cos(θ)
```

或展开计算：

```
a · b = a₁×b₁ + a₂×b₂ + ... + aₙ×bₙ
```

---

## 记忆要点

| 特征 | 内容 |
|------|------|
| **输入** | 两个向量 |
| **输出** | **一个数值**（标量） |
| **行向量 × 列向量** | `[a₁, a₂] × [b₁] = a₁×b₁ + a₂×b₂` |
| **几何意义** | 投影 × 长度 |

---

## 例子（2维）

```
a = [1, 2]
b = [3, 4]

a · b = 1×3 + 2×4 = 3 + 8 = 11
```

---

## 几何意义

```
内积 = |a| × |b| × cos(θ)

|a|, |b| = 向量长度
θ = 两向量夹角
```

| 内积值 | 夹角 | 含义 |
|--------|------|------|
| **> 0** | θ < 90° | 方向相似 |
| **= 0** | θ = 90° | 正交（垂直） |
| **< 0** | θ > 90° | 方向相反 |

---

## 直观理解

### 投影角度

```
内积 = a 在 b 方向上的投影 × b 的长度
```

想象 a 投射影子到 b 上，影子长度 × b 的长度 = 内积。

### 相似度角度

- 内积大 → 两向量方向相近
- 内积小 → 两向量方向相反
- 内积为0 → 两向量垂直

## 应用场景

| 领域 | 应用 |
|------|------|
| **机器学习** | 向量相似度计算（如词向量、嵌入向量） |
| **RAG** | 查询与文档的相似度匹配 |
| **推荐系统** | 用户向量与物品向量的匹配 |
| **神经网络** | 注意力机制中的 Query-Key 计算 |
| **物理学** | 功 = 力 × 位移 × cos(θ) |

---

## 在 AI 中的意义

### Embedding 相似度

```python
# 两个句子的嵌入向量
query = [0.2, 0.8, 0.1]
doc = [0.3, 0.7, 0.2]

# 内积越大，越相似
similarity = query · doc = 0.2×0.3 + 0.8×0.7 + 0.1×0.2 = 0.62
```

### 注意力机制

```
Attention(Q, K) = softmax(Q · Kᵀ / √d)
```

Query 和 Key 的内积决定注意力权重。

---

## 与外积对比

| 对比 | 内积 | 外积 |
|------|------|------|
| 符号 | `a · b` 或 `aᵀb` | `a ⊗ b` 或 `abᵀ` |
| 形式 | 行向量 × 列向量 | 列向量 × 行向量 |
| 输出 | **数值** | **矩阵** |
| 维度 | 降低 | 升高 |
