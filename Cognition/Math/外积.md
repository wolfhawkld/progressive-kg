# 外积（Outer Product）

> 两个向量之间的"组合生成"运算，结果是**一个矩阵**。

---

## 定义

对于两个向量 **a**（m维）和 **b**（n维）：

```
a ⊗ b = a × bᵀ
```

结果是一个 **m×n 矩阵**。

---

## 记忆要点

| 特征 | 内容 |
|------|------|
| **输入** | 两个向量 |
| **输出** | **一个矩阵** |
| **列向量 × 行向量** | `[a₁] × [b₁, b₂] = [a₁×b₁, a₁×b₂]` |
| `[a₂]` `[a₂×b₁, a₂×b₂]` | |
| **几何意义** | 方向信息的组合 |

---

## 例子（2维 × 2维）

```
a = [1, 2]ᵀ  (列向量)
b = [3, 4]   (行向量)

a ⊗ b = [1] × [3, 4]
        [2]

      = [1×3  1×4]
        [2×3  2×4]

      = [3   4 ]
        [6   8 ]
```

结果是一个 **2×2 矩阵**。

---

## 例子（3维 × 2维）

```
a = [1, 2, 3]ᵀ  (列向量，3维)
b = [4, 5]      (行向量，2维)

a ⊗ b = [1] × [4, 5]
        [2]
        [3]

      = [1×4  1×5]
        [2×4  2×5]
        [3×4  3×5]

      = [4   5 ]
        [8   10]
        [12  15]
```

结果是一个 **3×2 矩阵**。

---

## 几何意义

```
外积 = 向量 a 和向量 b 的"方向组合"
```

### 矩阵结构解读

```
         b₁        b₂
      ┌────────┬────────┐
   a₁ │ a₁×b₁  │ a₁×b₂  │
      ├────────┼────────┤
   a₂ │ a₂×b₁  │ a₂×b₂  │
      └────────┴────────┘
```

| 矩阵元素 | 含义 |
|----------|------|
| **对角线** | aᵢ × bᵢ，**同维度组合**（第i分量与第i分量） |
| **非对角线** | aᵢ × bⱼ，**跨维度组合**（第i分量与第j分量） |

### 特例：a ⊗ a（自己与自己）

```
a ⊗ a = [a₁] × [a₁, a₂] = [a₁²,  a₁×a₂]
        [a₂]             [a₂×a₁, a₂²]
```

此时对角线才是"自组合"（aᵢ²）。

### 物理意义

- 外积描述了两个向量的"方向信息"如何组合
- 矩阵的每个元素是两个分量方向的"耦合强度"
- 对角线：同一维度的匹配程度
- 非对角线：不同维度的交叉影响

---

## 为什么重要？

### 1. 记录方向信息

```
外积矩阵 = 向量的"方向指纹"

若某方向分量大 → 该方向在矩阵中占比大
```

### 2. 累积历史信息

```
A = I + Σ(aᵢ ⊗ aᵢ)  (累积多个外积)

作用:
  - 记录历史向量的方向分布
  - 哪个方向出现多 → 该方向在 A 中占比大
  - A ≈ 协方差矩阵
```

### 3. 从向量到矩阵的桥梁

```
向量运算 → 矩阵运算 → 更丰富的表达
```

---

## 在 LinUCB 中的应用

### 累积矩阵 A 的更新

```
每次收到反馈:
  A ← A + x ⊗ x = A + xxᵀ
  b ← b + r × x

含义:
  - xxᵀ 记录查询向量 x 的方向信息
  - 累积多次后，A 包含所有历史查询的方向分布
  - 不确定性 √(xᵀA⁻¹x) 可计算
```

### 为什么用外积？

```
内积: x · x = 一个数（只知"大小"）
外积: x ⊗ x = 矩阵（知"方向+分布"）

LinUCB 需要:
  - 知道哪些方向查询多 → 外积记录
  - 计算不确定性 → A⁻¹ 提供
```

## 应用场景

| 领域 | 应用 |
|------|------|
| **LinUCB** | 累积矩阵 A = Σ(xxᵀ)，记录查询方向分布 |
| **协方差矩阵** | Cov = E[(x-μ)(x-μ)ᵀ] = 外积的期望 |
| **量子力学** | 态矢量 → 密度矩阵 |
| **张量分解** | 外积是张量积的特例 |
| **图神经网络** | 邻接矩阵的构造 |

## 与内积对比

| 对比 | 内积 | 外积 |
|------|------|------|
| 符号 | `a · b` 或 `aᵀb` | `a ⊗ b` 或 `abᵀ` |
| 形式 | 行向量 × 列向量 | 列向量 × 行向量 |
| 输出 | **数值** | **矩阵** |
| 维度变化 | 降低 | 升高 |
| 信息量 | 只知相似度 | 知方向分布 |

```
内积: [a₁, a₂] × [b₁] = 数值（信息压缩）
                 [b₂]

外积: [a₁] × [b₁, b₂] = 矩阵（信息扩展）
      [a₂]
```

## 直观类比

### 内积 = 影子

```
a 在 b 上的投影长度 → 内积
（压缩成一个数）
```

### 外积 = 网格

```
a 的每个分量与 b 的每个分量组合 → 外积矩阵
（扩展成一个网格/矩阵）
```
