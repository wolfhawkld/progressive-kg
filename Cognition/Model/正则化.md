# 正则化

## 概念

正则化是防止过拟合的技术，通过限制模型复杂度或增加训练难度，提升模型泛化能力。

## 主要方法

### Dropout

```python
# 训练时
y = x * mask / (1 - p)  # mask随机置0，缩放保持期望

# 推理时
y = x  # 使用全部神经元
```

- **原理**: 随机失活神经元，相当于集成多个子网络
- **典型比例**: 0.1 ~ 0.5
- **变体**: 
  - DropConnect: 随机失活权重
  - Spatial Dropout: 整个特征图失活
  - DropPath: 随机失活整条路径

### Weight Decay (权重衰减)

```
L_total = L_task + λ * ||W||²
```

- **L2正则化**: 权重平方和
- **L1正则化**: 权重绝对值和，产生稀疏解
- **AdamW**: 将weight decay与梯度更新解耦

### 早停 (Early Stopping)

- 监控验证集损失
- 性能不再提升时停止训练
- **实现**: 保存最佳模型，patience参数控制容忍度

### 数据增强

**图像**:
- 几何变换: 翻转、旋转、裁剪、缩放
- 颜色变换: 亮度、对比度、饱和度
- Mixup: 两张图像混合
- CutMix: 图像区域替换
- AutoAugment: 自动搜索增强策略

**文本**:
- 同义词替换
- 回译
- 随机插入/删除/交换

### Label Smoothing

```
y_smooth = (1 - ε) * y_onehot + ε / K
```

- 防止模型过度自信
- 典型ε = 0.1

## 正则化效果对比

| 方法 | 参数量 | 训练速度 | 推理开销 | 适用场景 |
|-----|--------|---------|---------|---------|
| Dropout | 0 | 微增 | 无 | 全连接层 |
| Weight Decay | 0 | 无 | 无 | 通用 |
| 早停 | 0 | 无 | 无 | 通用 |
| 数据增强 | 0 | 增加 | 无 | 数据有限 |
| Mixup/CutMix | 0 | 微增 | 无 | 图像分类 |

## 与其他技术配合

- **BatchNorm + Dropout**: 注意顺序，BN在前
- **大模型**: 倾向于减少Dropout，增加数据量
- **Transformer**: 通常在FFN和注意力输出后加Dropout
