# 感受野

## 概念

感受野（Receptive Field）指卷积神经网络中某一层输出特征图上的一个像素点，对应输入图像上的区域大小。感受野越大，能捕获的全局信息越多。

## 计算

### 理论感受野

```
RF_l = RF_{l-1} + (k_l - 1) * ∏_{i=1}^{l-1} s_i
```

- RF_l: 第l层的感受野
- k_l: 第l层的卷积核大小
- s_i: 第i层的步长

### 递推公式

```
RF_1 = k_1  # 第一层感受野等于卷积核大小
RF_l = RF_{l-1} + (k_l - 1) * jump_{l-1}
jump_l = jump_{l-1} * s_l
```

### 示例

3个3×3卷积层，步长均为1：
```
RF_1 = 3
RF_2 = 3 + (3-1)*1 = 5
RF_3 = 5 + (3-1)*1 = 7
```

等效于1个7×7卷积核，但参数量更少（3×3×3=27 vs 7×7=49）。

## 有效感受野

- **定义**: 输出对输入各位置的实际敏感度
- **特点**: 呈高斯分布，中心区域贡献更大
- **原因**: 权重分布、非线性激活

**启示**: 理论感受野大，但有效感受野可能小得多。

## 扩大感受野的方法

### 1. 更深的网络
- 堆叠更多卷积层
- 感受野线性增长

### 2. 更大的卷积核
- 直接增大k
- 参数量二次增长

### 3. 池化层
- 下采样增大后续层的"跳跃"步长
- 信息有损

### 4. 空洞卷积 (Dilated Convolution)

```
卷积核间隔扩张，不增加参数
```

- 空洞率d: 卷积核元素间隔d-1个零
- 有效卷积核大小: k' = k + (k-1)(d-1)
- **应用**: DeepLab语义分割

### 5. 全局池化
- Global Average Pooling
- 整张图作为感受野

## 各方法对比

| 方法 | 参数增长 | 感受野增长 | 信息保留 |
|-----|---------|-----------|---------|
| 加深网络 | 线性 | 线性 | 好 |
| 大卷积核 | 二次 | 线性 | 好 |
| 池化 | 无 | 指数 | 有损 |
| 空洞卷积 | 无 | 指数 | 好 |

## 任务需求

| 任务 | 感受野需求 |
|-----|-----------|
| 图像分类 | 全图（最终层） |
| 目标检测 | 大物体需要大感受野 |
| 语义分割 | 大感受野，多尺度 |
| 边缘检测 | 小感受野即可 |
