# 残差连接

## 概念

残差连接（Residual Connection）让网络层学习残差映射而非直接映射，解决了深层网络的训练退化问题。是现代深度网络的标配组件。

## 核心公式

```
y = F(x) + x
```

- **x**: 输入（恒等映射）
- **F(x)**: 残差函数，网络层学习的目标
- **y**: 输出

## 为什么有效

### 梯度流动

```
∂y/∂x = ∂F(x)/∂x + 1
```

- 梯度至少为1，不会消失
- 形成"梯度高速公路"，浅层梯度直达

### 优化难度

- 恒等映射是最简单的解
- 学习F(x)=0比学习恒等映射更容易
- 网络可以"选择性"学习需要的变化

### 集成效应

- 等效于指数数量子网络的集成
- 不同路径组合

## 主要变体

### 标准残差

```
y = F(x) + x
```

### Pre-activation ResNet

```
y = x + F(Norm(Act(x)))  # BN和ReLU在卷积前
```

- 更好的梯度流动
- 训练更稳定

### Dense Connection (DenseNet)

```
y = Concat([x_1, x_2, ..., x_l])
```

- 密集连接，所有前置层输出拼接
- 特征复用，参数高效

### Highway Network

```
y = T * F(x) + (1 - T) * x  # T为门控
```

- 可学习的残差比例

## 应用场景

| 架构 | 残差形式 |
|-----|---------|
| ResNet | 标准残差 |
| Transformer | Post-Norm/Pre-Norm |
| U-Net | 跳跃连接 |
| DenseNet | 密集连接 |
| BERT/GPT | Pre-Norm残差 |

## Transformer中的残差

```
# Pre-Norm (现代选择)
x = x + Attention(LayerNorm(x))
x = x + FFN(LayerNorm(x))

# Post-Norm (原始)
x = LayerNorm(x + Attention(x))
x = LayerNorm(x + FFN(x))
```

## 注意事项

- **维度匹配**: F(x)和x维度必须一致，否则需要投影
- **加法位置**: 通常在激活函数之前
- **内存开销**: 需要保存输入用于反向传播
