# 位置编码

## 概念

Transformer的自注意力机制本身不具备位置感知能力（排列不变），需要显式注入位置信息。位置编码就是为序列中每个位置生成唯一表示。

## 主要类型

### Sinusoidal位置编码 (原始Transformer)

```
PE(pos, 2i) = sin(pos / 10000^(2i/d))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d))
```

- **特点**: 固定编码，不需要学习
- **优点**: 可以外推到训练时未见过的长度
- **原理**: 不同频率的正弦/余弦组合，位置关系可通过线性变换表示

### 可学习位置编码

```
PE = Embedding(max_seq_len, d_model)  # 可训练
```

- **特点**: 位置嵌入作为参数学习
- **优点**: 适应特定任务
- **缺点**: 长度外推能力弱
- **应用**: BERT、GPT-2

### 相对位置编码

**核心思想**: 编码位置间的相对距离，而非绝对位置

**代表方法**:
- **Shaw's RPE**: 在注意力分数中加入相对位置偏置
- **T5 Bias**: 添加可学习的相对位置偏置矩阵
- **ALiBi**: 线性偏置，根据距离线性衰减注意力

```
Attention = QK^T + bias  # bias由相对位置决定
```

### RoPE (Rotary Position Embedding)

**核心思想**: 将位置信息编码为旋转矩阵，作用于Q和K

```
f(x, m) = x * e^(imθ)  # 复数形式旋转
```

- **特点**: 结合绝对和相对位置优点
- **优点**: 
  - 自然表达相对位置
  - 长度外推能力强
  - 无需额外参数
- **应用**: LLaMA、GLM、PaLM
- **详见**: [[RoPE]]

### ALiBi (Attention with Linear Biases)

```
Attention(i, j) = q_i · k_j - m · |i - j|
```

- **特点**: 根据距离线性惩罚注意力分数
- **优点**: 极强的长度外推能力
- **应用**: BLOOM、MPT

## 长度外推对比

| 方法 | 外推能力 | 参数量 | 适用场景 |
|-----|---------|--------|---------|
| Sinusoidal | 一般 | 0 | 通用 |
| 可学习 | 弱 | O(L*d) | 固定长度任务 |
| RoPE | 强 | 0 | LLM标配 |
| ALiBi | 很强 | O(H) | 超长序列 |

*L: 最大长度, d: 维度, H: 注意力头数*

## 1D vs 2D位置编码

**1D序列**:
- 文本、时间序列
- RoPE、ALiBi常用

**2D图像**:
- ViT需要编码空间位置
- 方法: 
  - 两个1D编码拼接
  - 2D正弦编码
  - 可学习2D嵌入
