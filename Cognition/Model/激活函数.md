# 激活函数

## 概念

激活函数为神经网络引入非线性，使网络能够学习复杂的模式。没有激活函数，多层神经网络等价于单层线性变换。

## 常见激活函数

### ReLU (Rectified Linear Unit)
```
f(x) = max(0, x)
```
- **优点**: 计算简单，缓解梯度消失，稀疏激活
- **缺点**: Dead ReLU问题（神经元永不被激活）
- **应用**: 深度网络默认选择

### Sigmoid
```
f(x) = 1 / (1 + e^(-x))
```
- **优点**: 输出(0,1)，适合概率输出
- **缺点**: 梯度消失，输出非零中心
- **应用**: 二分类输出层、注意力权重

### Tanh
```
f(x) = (e^x - e^(-x)) / (e^x + e^(-x))
```
- **优点**: 输出(-1,1)，零中心
- **缺点**: 梯度消失
- **应用**: RNN隐藏状态

### GELU (Gaussian Error Linear Unit)
```
f(x) = x * Φ(x)  # Φ为标准正态CDF
```
- **优点**: 平滑、非单调，性能优于ReLU
- **应用**: Transformer、BERT、GPT系列

### Swish / SiLU
```
f(x) = x * sigmoid(x)
```
- **优点**: 平滑、有下界、非单调
- **应用**: 现代网络架构

### Leaky ReLU / PReLU
```
f(x) = max(αx, x)  # α通常为0.01
```
- **优点**: 解决Dead ReLU
- **应用**: 需要保留负信息的场景

## 选择原则

| 场景 | 推荐激活函数 |
|------|-------------|
| 隐藏层（通用） | ReLU / GELU |
| 输出层（二分类） | Sigmoid |
| 输出层（多分类） | Softmax |
| 输出层（回归） | 无 / Linear |
| RNN/LSTM | Tanh |
| Transformer | GELU |
