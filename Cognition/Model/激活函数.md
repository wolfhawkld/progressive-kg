# 激活函数

> 对神经元预激活施加非线性或门控，使多层网络不再退化为单个仿射映射。

## 核心作用

若各层只有线性变换和 bias，多层复合仍是一个仿射映射。非线性激活使网络能够构造分段或平滑的非线性函数；它的输出范围、导数、均值和计算成本也会影响优化。

- 表达层面：打破仿射复合的闭包。
- 优化层面：改变局部 Jacobian 和激活统计。

## 常见隐藏层激活

| 函数 | 公式 | 主要边界 |
|---|---|---|
| ReLU | $\max(0,x)$ | 正区梯度简单，负区为 0；单元可能长期处于不激活区 |
| Leaky ReLU | $\max(\alpha x,x)$ | 负区保留固定小斜率；$\alpha$ 需选择 |
| PReLU | 同上但 $\alpha$ 可学习 | 增加参数并可能过拟合小数据 |
| GELU | $x\Phi(x)$ | 平滑门控，常见于 Transformer；不保证普遍优于 ReLU |
| SiLU/Swish | $x\sigma(x)$ | 平滑、非单调，常见于现代 CNN/LLM |
| tanh | $\tanh(x)$ | 有界且零中心，但大绝对值时饱和 |
| sigmoid | $1/(1+e^{-x})$ | 输出 $(0,1)$，常用于门；饱和且非零中心 |

ReLU 的“dead”并非数学上永远无法恢复：后续参数或输入分布变化可能让预激活回到正区；但如果所有相关样本梯度都为零，优化器很难直接恢复该单元。

## 输出层不是固定配方

| 任务参数化 | 常见输出/损失组合 | 注意 |
|---|---|---|
| 二元或多标签概率 | logits + BCE-with-logits | 训练时不要手工 sigmoid 后再交给带 logits 版本 |
| 互斥多类 | logits + cross-entropy | 框架通常内部做 log-softmax |
| 无界回归均值 | 线性输出 | 仍需匹配噪声与损失假设 |
| 正值参数 | softplus / exp 等 | 数值范围和尾部行为不同 |

Softmax 是向量级归一化映射，不是普通逐元素隐藏激活。注意力权重通常由 Softmax 产生，而 sigmoid 更常见于独立门控或多标签输出。

## 选择原则

- 与初始化和归一化共同考虑激活方差。
- 检查低精度下的溢出、下溢和近似 kernel。
- 门控 FFN（GLU/SwiGLU）还会改变中间宽度与参数预算，不能只比较激活名称。
- 用目标模型和训练预算做消融；“Transformer 用 GELU”“CNN 用 ReLU”只是常见起点。

## 关系网络

- 参与 [[反向传播]] — 局部导数进入 VJP
- 影响 [[梯度消失与梯度爆炸]] — 饱和、零导数和尺度改变 Jacobian
- 配合 [[归一化层]] — 归一化位置与激活顺序共同决定分布
- 应用 [[Transformer架构]] — FFN 常使用 GELU、SiLU 或门控变体
- 应用 [[循环神经网络]] — tanh/sigmoid 常用于状态与门控
