# 扩散模型

## 概念

扩散模型（Diffusion Model）通过逐步去噪过程生成高质量样本。在图像生成、视频生成等领域取得突破性成果。

## 核心思想

### 前向扩散（加噪）

```
x_t = √(α_t) * x_0 + √(1 - α_t) * ε
```

- 从干净数据x_0逐步添加高斯噪声
- 经过T步后，x_T接近纯噪声
- 过程固定，不需要学习

### 反向去噪（生成）

```
学习 p_θ(x_{t-1} | x_t)
```

- 从噪声x_T逐步恢复x_0
- 需要神经网络预测噪声
- 这是需要训练的部分

## 训练目标

```
L = E_t,x_0,ε [ || ε - ε_θ(x_t, t) ||² ]
```

- ε: 真实添加的噪声
- ε_θ: 网络预测的噪声
- 目标: 最小化噪声预测误差

## 网络架构

### U-Net

- 编码器-解码器结构
- 跳跃连接保留细节
- 时间步嵌入注入时间信息

### 时间步嵌入

```python
t_emb = SinusoidalPositionEmbedding(t)
t_emb = MLP(t_emb)
```

- 告诉网络当前去噪到哪一步
- 类似Transformer的位置编码

### 注意力机制

- Self-Attention捕获全局依赖
- Cross-Attention引入条件信息（如文本）

## 条件生成

### 分类器引导 (Classifier Guidance)

```
在去噪过程中加入类别梯度
```

### 无分类器引导 (Classifier-Free Guidance, CFG)

```
ε_pred = ε_uncond + s * (ε_cond - ε_uncond)
```

- s: 引导强度
- 训练时随机drop条件
- 推理时混合有条件和无条件预测

## 重要变体

### DDPM (Denoising Diffusion Probabilistic Models)
- 奠基工作
- 固定方差，预测噪声

### DDIM (Denoising Diffusion Implicit Models)
- 非马尔可夫采样
- 加速生成，步数可调

### Latent Diffusion (Stable Diffusion)
- 在低维潜空间扩散
- 大幅降低计算成本
- VAE编码图像 → 潜空间扩散 → VAE解码生成

### DiT (Diffusion Transformer)
- 用Transformer替代U-Net
- 扩展性好，适合大规模

## 与其他生成模型对比

| 特性 | 扩散模型 | GAN | VAE |

| 训练稳定性 | 高 | 低 | 高 |
