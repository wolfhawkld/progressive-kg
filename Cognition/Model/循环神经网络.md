# 循环神经网络 (RNN)

## 概念

循环神经网络处理序列数据，每个时间步的输出依赖当前输入和前一时刻的隐藏状态，具有记忆能力。

## 基本RNN

```
h_t = tanh(W_hh * h_{t-1} + W_xh * x_t + b_h)
y_t = W_hy * h_t + b_y
```

- **问题**: 梯度消失/爆炸，难以捕获长距离依赖

## LSTM (Long Short-Term Memory)

### 结构

```
遗忘门: f_t = σ(W_f * [h_{t-1}, x_t] + b_f)
输入门: i_t = σ(W_i * [h_{t-1}, x_t] + b_i)
候选记忆: C̃_t = tanh(W_C * [h_{t-1}, x_t] + b_C)
记忆更新: C_t = f_t * C_{t-1} + i_t * C̃_t
输出门: o_t = σ(W_o * [h_{t-1}, x_t] + b_o)
隐藏状态: h_t = o_t * tanh(C_t)
```

### 关键创新

- **门控机制**: 控制信息流动
- **记忆单元**: 长期记忆存储
- **解决**: 梯度消失，捕获长距离依赖

## GRU (Gated Recurrent Unit)

### 结构

```
重置门: r_t = σ(W_r * [h_{t-1}, x_t])
更新门: z_t = σ(W_z * [h_{t-1}, x_t])
候选隐藏: h̃_t = tanh(W * [r_t * h_{t-1}, x_t])
隐藏状态: h_t = (1 - z_t) * h_{t-1} + z_t * h̃_t
```

### 对比LSTM

| 特性 | LSTM | GRU |
|-----|------|-----|
| 门数量 | 3个 | 2个 |
| 参数量 | 多 | 少 |
| 性能 | 略优 | 接近 |
| 训练速度 | 慢 | 快 |

## RNN变体

### 双向RNN (BiRNN)
- 正向和反向两个RNN
- 同时利用过去和未来信息
- **应用**: NER、序列标注

### 深层RNN
- 多层RNN堆叠
- 每层提取不同抽象级别的特征

### Seq2Seq (Encoder-Decoder)
- 编码器编码输入序列
- 解码器生成输出序列
- **应用**: 机器翻译、文本摘要

### Attention增强RNN
- 在Seq2Seq基础上加入注意力
- 解决长序列信息瓶颈
- **代表作**: Bahdanau Attention

## 训练技巧

### Teacher Forcing
- 训练时使用真实标签作为下一输入
- 加速收敛，但可能导致exposure bias

### Scheduled Sampling
- 训练时混合使用真实标签和模型预测
- 缓解exposure bias

### 梯度裁剪
- 防止梯度爆炸
- `grad = grad * min(1, threshold/||grad||)`

## 与Transformer对比

| 特性 | RNN | Transformer |
|-----|-----|-------------|
| 并行性 | 无法并行 | 完全并行 |
| 长距离依赖 | LSTM缓解 | 直接建模 |
| 计算复杂度 | O(n) | O(n²) |
| 训练效率 | 低 | 高 |
