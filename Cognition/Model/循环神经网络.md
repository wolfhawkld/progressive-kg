# 循环神经网络 (RNN)

> 循环神经网络处理序列数据，每个时间步的输出依赖当前输入和前一时刻的隐藏状态，具有记忆能力。

基本RNN

```
h_t = tanh(W_hh * h_{t-1} + W_xh * x_t + b_h)
y_t = W_hy * h_t + b_y
```

- **问题**: 梯度消失/爆炸，难以捕获长距离依赖

## LSTM (Long Short-Term Memory)

LSTM 把 cell state 与 hidden state 分开，并用门控制保留、写入和输出的信息。

### 结构

```
遗忘门: f_t = σ(W_f * [h_{t-1}, x_t] + b_f)
输入门: i_t = σ(W_i * [h_{t-1}, x_t] + b_i)
候选记忆: C̃_t = tanh(W_C * [h_{t-1}, x_t] + b_C)
记忆更新: C_t = f_t * C_{t-1} + i_t * C̃_t
输出门: o_t = σ(W_o * [h_{t-1}, x_t] + b_o)
隐藏状态: h_t = o_t * tanh(C_t)
```

### 关键创新

- **门控机制**: 控制信息流动
- **记忆单元**: 长期记忆存储
- **作用**: 加性记忆路径和门控能缓解梯度消失、改善长距离依赖，但不能保证任意长度上都保留信息

## GRU (Gated Recurrent Unit)

GRU 将记忆和输出合并在单一 hidden state 中，以较少门控实现与 LSTM 相似的增量状态更新。

### 结构

```
重置门: r_t = σ(W_r * [h_{t-1}, x_t])
更新门: z_t = σ(W_z * [h_{t-1}, x_t])
候选隐藏: h̃_t = tanh(W * [r_t * h_{t-1}, x_t])
隐藏状态: h_t = (1 - z_t) * h_{t-1} + z_t * h̃_t
```

### 对比LSTM

| 特性 | LSTM | GRU |
|-----|------|-----|
| 门数量 | 3个 | 2个 |
| 参数量 | 多 | 少 |
| 性能 | 依任务和调参而异 | 依任务和调参而异，没有统一优劣 |
| 单步成本 | 门和状态较多 | 通常参数与运算较少，但实际速度取决于 kernel 和硬件 |

## RNN变体

变体主要改变可见上下文方向、堆叠深度或 Encoder–Decoder 之间的信息瓶颈。

### 双向RNN (BiRNN)
- 正向和反向两个RNN
- 同时利用过去和未来信息
- **应用**: NER、序列标注

### 深层RNN
- 多层RNN堆叠
- 每层提取不同抽象级别的特征

### Seq2Seq (Encoder-Decoder)
- 编码器编码输入序列
- 解码器生成输出序列
- **应用**: 机器翻译、文本摘要

### Attention增强RNN
- 在Seq2Seq基础上加入注意力
- 解决长序列信息瓶颈
- **代表作**: Bahdanau Attention

## 训练技巧

序列训练除了优化器外，还要处理训练输入与推理输入不一致以及长时间步梯度尺度。

### Teacher Forcing
- 自回归训练时使用真实的前一目标 token，而不是模型上一时刻的预测
- 加速收敛，但可能导致exposure bias

### Scheduled Sampling
- 训练时混合使用真实标签和模型预测
- 缓解exposure bias

### 梯度裁剪
- 防止梯度爆炸
- `grad = grad * min(1, threshold/||grad||)`

## 与Transformer对比

| 特性 | RNN | Transformer |
|-----|-----|-------------|
| 时间步并行性 | 隐状态递归，时间步难以并行；批和矩阵维仍可并行 | 训练时位置可并行；自回归生成仍逐步进行 |
| 长距离依赖 | LSTM/GRU 缓解，但信息仍沿状态传递 | 全局注意力可直接连接位置，但效果仍受层数、注意力模式和训练影响 |
| 序列长度项 | 递归扫描为线性 | 全局注意力为二次；局部/稀疏变体例外 |
| 适用权衡 | 流式状态紧凑、低延迟场景仍有价值 | 大规模并行训练和全局交互更有优势 |
