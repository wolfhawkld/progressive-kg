# 损失函数

## 概念

损失函数衡量模型预测与真实值之间的差距，是训练优化的目标。选择合适的损失函数对模型性能至关重要。

## 回归损失

### MSE (Mean Squared Error)
```
L = (1/n) * Σ(y - ŷ)²
```
- **特点**: 对大误差惩罚更强，对异常值敏感
- **应用**: 一般回归任务

### MAE (Mean Absolute Error)
```
L = (1/n) * Σ|y - ŷ|
```
- **特点**: 对异常值鲁棒，零点不可导
- **应用**: 需要鲁棒性的回归任务

### Huber Loss
```
L = { 0.5(y-ŷ)²        if |y-ŷ| ≤ δ
    { δ|y-ŷ| - 0.5δ²   otherwise
```
- **特点**: 结合MSE和MAE优点
- **应用**: 异常值较多的回归

## 分类损失

### 交叉熵损失 (Cross-Entropy)
```
L = -Σ y_i * log(p_i)
```
- **二分类**: Binary Cross-Entropy
- **多分类**: Categorical Cross-Entropy
- **应用**: 分类任务标配

### Focal Loss
```
L = -α(1-p)^γ * log(p)
```
- **特点**: 降低易分类样本权重，聚焦困难样本
- **应用**: 类别不平衡、目标检测

## 生成模型损失

### 对比损失 (Contrastive Loss)
- 正样本拉近，负样本推远
- **应用**: 自监督学习、表征学习

### KL散度
```
KL(P||Q) = Σ P(x) * log(P(x)/Q(x))
```
- **应用**: VAE、知识蒸馏

### 对抗损失 (Adversarial Loss)
- GAN中生成器与判别器的博弈
- **应用**: 生成对抗网络

## 排序损失

### Triplet Loss
```
L = max(0, d(a,p) - d(a,n) + margin)
```
- **应用**: 人脸识别、检索

### Contrastive Loss (Siamese)
```
L = (1-y)*d² + y*max(0, margin-d)²
```
- **应用**: 孪生网络

## 选择原则

| 任务类型 | 推荐损失函数 |
|---------|-------------|
| 回归（无异常） | MSE |
| 回归（有异常） | MAE / Huber |
| 二分类 | BCE |
| 多分类 | Cross-Entropy |
| 类别不平衡 | Focal Loss |
| 生成任务 | 对抗损失 + KL散度 |
| 表征学习 | 对比损失 / Triplet Loss |
