# 损失函数

> 将预测、目标与约束映射为可优化标量，是任务代价或统计假设的训练代理。

## 核心边界

损失函数定义单样本或小批预测的训练代价；经验风险是其在训练数据上的平均，训练目标还可能加入正则项。评估指标、真实业务效用和训练损失可以相同，也可以不同。

$$
\min_\theta\frac1n\sum_{i=1}^{n}
\ell(f_\theta(x_i),y_i)+\lambda R(\theta)
$$

可微性便于梯度优化，但不是损失函数存在的必要条件；实践中常用可微 surrogate 近似不可微的排名、0–1 错误或业务成本。

回归损失

| 损失 | 形式 | 适用假设/权衡 |
|---|---|---|
| MSE | $(y-\hat y)^2$ | 强调大残差；与同方差高斯噪声似然相连 |
| MAE | $|y-\hat y|$ | 对大残差较稳健；零点不可微但可用次梯度 |
| Huber | 小残差二次、大残差线性 | 阈值 $\delta$ 决定平滑与稳健权衡 |
| Quantile | 非对称绝对误差 | 估计条件分位数，而非条件均值 |

“数据有异常值就用 MAE”仍过于简单：噪声分布、目标统计量、校准和业务代价都应参与选择。

## 分类损失

对单标签类别 $y$，交叉熵可写为：

$$
\ell=-\log p_\theta(y\mid x)
$$

实现通常直接接收 logits 并内部使用 log-sum-exp，以避免先 Softmax 后取对数的数值问题。BCE 适用于独立二元/多标签目标，多类交叉熵适用于互斥类别；标签编码和输出层必须匹配。

Focal Loss 在交叉熵上乘 $(1-p_t)^\gamma$，降低已高置信正确样本的权重。它可帮助某些类别不平衡检测任务，但 $\alpha,\gamma$ 需要验证，不能替代采样、阈值和校准分析。

## 分布、生成与表示学习

- **KL divergence** 比较两个分布且有方向性，$D_{KL}(P\|Q)\ne D_{KL}(Q\|P)$；它常作为 VAE 或蒸馏目标的一部分。
- **对比损失** 依赖正负样本、温度和 batch 构造；InfoNCE、pairwise contrastive、triplet loss 不是同一个公式。
- **对抗目标** 定义生成器与判别器的博弈；并非所有生成模型都同时需要 KL。
- **扩散目标** 可预测噪声、score、$x_0$ 或 velocity，并采用不同时间权重。

## 选择与验证

1. 明确训练目标希望估计均值、概率、分位数还是排序。
2. 把类别/样本权重与真实决策成本对应起来。
3. 检查 reduction（sum/mean）、mask、ignore index 和 batch 组成。
4. 同时报告与业务一致的验证指标及不确定性。
5. 对复合损失记录每一项尺度和权重，防止某项仅因数值大而主导。
