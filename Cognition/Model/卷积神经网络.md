# 卷积神经网络 (CNN)

> 通过局部连接和参数共享提取空间特征，对离散平移具有等变性并可构建有限不变性。

卷积神经网络（CNN）在规则网格上复用局部滤波器。理想条件下，输入平移会使特征图相应平移，即“平移等变”；分类网络可通过池化、全局聚合和数据增强获得一定的平移不变性，但这不是普通卷积自动保证的绝对性质。

核心组件

典型 CNN 由局部卷积、下采样和任务输出头组成，现代架构会按任务省略或替换其中部分组件。

### 卷积层 (Convolution)

```
输出尺寸 = (输入尺寸 - 卷积核尺寸 + 2*填充) / 步长 + 1
```

- **卷积核 (Kernel/Filter)**: 提取特征的权重矩阵
- **步长 (Stride)**: 卷积核移动步长
- **填充 (Padding)**: 边界补零，控制输出尺寸
- **通道 (Channels)**: 深度方向的特征图数量

### 池化层 (Pooling)

- **最大池化**: 取窗口内最大值，保留显著特征
- **平均池化**: 取窗口内平均值
- **作用**: 下采样、增大后续层感受野，并提高对局部位移的鲁棒性；不保证任意平移下输出完全不变

### 全连接层 (Fully Connected)

- 特征展平后分类
- 现代网络常用全局平均池化替代

## 关键概念

感受野、dilation、groups 和转置映射分别描述上下文范围、采样方式、通道连接与尺寸变换。

### 感受野 (Receptive Field)

- 定义: 输出特征图上一个像素对应输入图像的区域大小
- **权衡**: 更大感受野能利用更广上下文，但也可能损失局部细节并增加计算；理论感受野大也不代表每个位置贡献相同
- **计算**: 逐层累积，考虑步长和卷积核大小

### 空洞卷积 (Dilated/Atrous Convolution)

```
卷积核间隔扩张，不增加参数量扩大感受野
```

- **应用**: 语义分割、需要大感受野的任务
- **空洞率 (Dilation Rate)**: 间隔大小

### 分组卷积 (Grouped Convolution)

- 将通道分组，每组独立卷积
- **深度可分离卷积**: 先做逐通道卷积（`groups=C_in`），再用 $1\times1$ 逐点卷积混合通道；它是分组卷积的特殊组合
- **优点**: 减少参数量和计算量

### 转置卷积 (Transposed Convolution)

- 常被俗称“反卷积”，但通常不是卷积的数学逆运算；它是普通卷积对输入的转置线性映射
- **应用**: 语义分割、生成模型

## 经典架构

| 网络 | 年份 | 创新 |
|-----|------|------|
| LeNet | 1998 | 卷积+池化架构 |
| AlexNet | 2012 | ReLU、Dropout、GPU训练 |
| VGG | 2014 | 小卷积核堆叠(3x3) |
| GoogLeNet | 2014 | Inception模块 |
| ResNet | 2015 | 残差连接 |
| DenseNet | 2017 | 密集连接 |
| EfficientNet | 2019 | 复合缩放 |

## 参数计算

**卷积层参数量**:
```
params = (K_h * K_w * C_in) * C_out + C_out
```

**卷积层乘加次数（groups = 1）**:
```
MACs = K_h * K_w * C_in * C_out * H_out * W_out
```

有些工具把一次乘加记为 1 MAC，有些把乘法和加法记为 2 FLOPs；比较结果前必须确认计数约定。分组卷积还要按组数缩减输入通道连接数。

## 与其他架构对比

| 特性 | CNN | Transformer |
|-----|-----|-------------|
| 归纳偏置 | 局部性、参数共享、平移等变 | 通常较弱，可通过位置和结构设计注入 |
| 感受野 | 需深层扩大 | 全局（自注意力） |
| 位置数复杂度 | 固定局部核通常近似线性 | 全局自注意力通常二次；稀疏/局部注意力例外 |
| 数据效率 | 视觉局部先验常有帮助 | 取决于预训练规模、增强与架构，不能一概而论 |
