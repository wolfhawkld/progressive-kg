# 卷积神经网络 (CNN)

## 概念

卷积神经网络通过卷积操作提取局部特征，具有参数共享和平移不变性。在图像处理领域取得突破性成果。

## 核心组件

### 卷积层 (Convolution)

```
输出尺寸 = (输入尺寸 - 卷积核尺寸 + 2*填充) / 步长 + 1
```

- **卷积核 (Kernel/Filter)**: 提取特征的权重矩阵
- **步长 (Stride)**: 卷积核移动步长
- **填充 (Padding)**: 边界补零，控制输出尺寸
- **通道 (Channels)**: 深度方向的特征图数量

### 池化层 (Pooling)

- **最大池化**: 取窗口内最大值，保留显著特征
- **平均池化**: 取窗口内平均值
- **作用**: 降维、增大感受野、提供平移不变性

### 全连接层 (Fully Connected)

- 特征展平后分类
- 现代网络常用全局平均池化替代

## 关键概念

### 感受野 (Receptive Field)

- 定义: 输出特征图上一个像素对应输入图像的区域大小
- **越大越好**: 感受野越大，捕获的全局信息越多
- **计算**: 逐层累积，考虑步长和卷积核大小

### 空洞卷积 (Dilated/Atrous Convolution)

```
卷积核间隔扩张，不增加参数量扩大感受野
```

- **应用**: 语义分割、需要大感受野的任务
- **空洞率 (Dilation Rate)**: 间隔大小

### 分组卷积 (Grouped Convolution)

- 将通道分组，每组独立卷积
- **深度可分离卷积**: 分组卷积 + 逐点卷积
- **优点**: 减少参数量和计算量

### 转置卷积 (Transposed Convolution)

- 又称反卷积，用于上采样
- **应用**: 语义分割、生成模型

## 经典架构

| 网络 | 年份 | 创新 |
|-----|------|------|
| LeNet | 1998 | 卷积+池化架构 |
| AlexNet | 2012 | ReLU、Dropout、GPU训练 |
| VGG | 2014 | 小卷积核堆叠(3x3) |
| GoogLeNet | 2014 | Inception模块 |
| ResNet | 2015 | 残差连接 |
| DenseNet | 2017 | 密集连接 |
| EfficientNet | 2019 | 复合缩放 |

## 参数计算

**卷积层参数量**:
```
params = (K_h * K_w * C_in) * C_out + C_out
```

**卷积层计算量 (FLOPs)**:
```
FLOPs = K_h * K_w * C_in * C_out * H_out * W_out
```

## 与其他架构对比

| 特性 | CNN | Transformer |
|-----|-----|-------------|
| 归纳偏置 | 平移不变性、局部性 | 弱 |
| 感受野 | 需深层扩大 | 全局（自注意力） |
| 计算复杂度 | 线性 | 二次 |
| 数据效率 | 高 | 低（需大规模数据） |
