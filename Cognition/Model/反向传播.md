# 反向传播

> 反向传播（Backpropagation）是神经网络训练的核心算法，通过链式法则从输出层向输入层逐层计算梯度，用于更新网络权重。

---

核心思想

1. **前向传播**: 输入 → 各层计算 → 输出 → 损失
2. **反向传播**: 损失对输出的梯度 → 逐层向后传递 → 各参数梯度
3. **参数更新**: 根据梯度调整权重

## 链式法则

```
∂L/∂w = ∂L/∂y * ∂y/∂w
```

对于多层网络:
```
∂L/∂w₁ = ∂L/∂y_n * ∂y_n/∂y_{n-1} * ... * ∂y₁/∂w₁
```

梯度通过计算图反向流动。

## 计算图示例

```
前向:
x → [Linear] → z → [ReLU] → a → [Linear] → y → [Loss] → L

反向:
∂L → ∂y → ∂a → ∂z → ∂w
```

## 关键概念

### 梯度流动
- 梯度从损失函数开始向后传递
- 每层的梯度依赖其后的所有层
- 激活函数的导数影响梯度强度

### 雅可比矩阵
- 多输出层的梯度传递涉及雅可比矩阵
- `∂L/∂x = (∂L/∂y) * (∂y/∂x)`

### 计算图优化
- 现代框架自动构建计算图
- 自动微分（Autograd）自动计算梯度
- 内存优化: 梯度检查点、重新计算

## 常见问题

### 梯度消失
- 连乘小于1的导数 → 梯度趋近于0
- 解决: ReLU、残差连接、归一化

### 梯度爆炸
- 连乘大于1的导数 → 梯度指数增长
- 解决: 梯度裁剪、权重初始化

## 反向传播在各层的实现

| 层类型 | 前向 | 反向 |
|-------|------|------|
| Linear | y = Wx + b | ∂W = ∂y·xᵀ, ∂x = Wᵀ·∂y |
| ReLU | y = max(0,x) | ∂x = ∂y * (x>0) |
| Softmax | y = exp(x)/Σexp(x) | ∂x = y·(∂y - Σ(y·∂y)) |
| BatchNorm | y = γ(x-μ)/σ + β | 复杂，框架自动处理 |

## 内存与计算权衡

- **存储激活**: 前向存储，反向使用
- **梯度检查点**: 只存部分激活，反向时重计算
- **权衡**: 用计算换内存
