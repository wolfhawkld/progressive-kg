# 梯度消失与梯度爆炸

> 深层或长时序计算中的 Jacobian 连乘使某些梯度方向快速衰减或放大，造成学习停滞或数值发散。

## 核心机制

对复合映射 $x_L=f_L\circ\cdots\circ f_1(x_0)$，反向传播包含 Jacobian 乘积：

$$
\frac{\partial L}{\partial x_0}
=\frac{\partial L}{\partial x_L}
J_LJ_{L-1}\cdots J_1
$$

若相关方向上的奇异值连乘长期小于 1，梯度会衰减；长期大于 1，则可能放大。矩阵方向会旋转并相互抵消，所以问题不能只用“每个标量导数是否小于 1”完整描述。

## 常见来源

- 饱和的 sigmoid/tanh 导数接近 0。
- 权重尺度或初始化使 Jacobian 奇异值偏离稳定范围。
- RNN 在长时间步上重复使用同一转移矩阵。
- 不稳定的损失、过大学习率或低精度溢出会放大爆炸症状。
- ReLU 的负半轴导数为 0，可能造成局部“dead unit”；正半轴导数为 1 也不表示整层 Jacobian 保长。

## 症状与诊断

| 现象 | 可能解释 | 还需排除 |
|---|---|---|
| 早期层梯度长期接近 0 | 消失或路径未激活 | 参数被冻结、`detach`、mask/数据错误 |
| 梯度范数突然极大 | 爆炸或异常 batch | 损失缩放、除零、数据离群值 |
| loss 为 NaN/Inf | 数值溢出 | 非法 log/sqrt、混合精度配置 |
| 层间尺度持续漂移 | 条件不良 | 归一化位置、初始化、残差缩放 |

```python
for name, p in model.named_parameters():
    if p.grad is not None:
        print(name, p.grad.detach().float().norm().item())
```

应观察多个 step 的层级分布、最大值和非有限比例，而不是只看一次总范数。

## 缓解方法

| 方法 | 主要作用 | 边界 |
|---|---|---|
| Xavier/He/正交初始化 | 控制初始化时前向与反向尺度 | 激活、宽度和训练更新会改变条件 |
| LSTM/GRU 门控 | 为长时序提供较稳定的加性记忆路径 | 不保证任意长依赖 |
| 残差连接 | Jacobian 中加入恒等路径 | $I+J_F$ 仍可抵消或放大 |
| BatchNorm/LN/RMSNorm | 控制激活尺度并改善优化条件 | 不构成梯度有界证明 |
| 梯度裁剪 | 限制爆炸更新的范数或数值 | 不能恢复已经消失的梯度；阈值过小会改变优化 |
| 学习率/损失缩放 | 避免更新过大或低精度下溢 | 与模型 Jacobian 问题相关但不等价 |

## 关系网络

- 机制 [[反向传播]] — Jacobian 链沿计算图反向相乘
- 受 [[激活函数]] — 饱和区、零导数和尺度影响局部 Jacobian
- 缓解 [[残差连接]] — 提供直接加性路径但无绝对下界
- 缓解 [[归一化层]] — 控制激活尺度和数值条件
- 诊断 [[训练调试]] — 需要结合梯度、激活、数据和非有限值监控
- 关联 [[奇异值]] — 奇异值描述局部线性映射对不同方向的放缩
