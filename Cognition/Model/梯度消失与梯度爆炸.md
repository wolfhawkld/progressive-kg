# 梯度消失与梯度爆炸

## 概念

深度神经网络训练中，梯度通过反向传播逐层传递时，可能出现梯度趋近于零（消失）或指数增长（爆炸）的问题，导致深层网络难以训练。

## 梯度消失 (Vanishing Gradient)

### 原因

```
∂L/∂x₁ = ∂L/∂x_n * Π(∂x_i/∂x_{i-1})
```

当激活函数导数|∂x/∂x'| < 1时，连乘导致梯度指数级衰减。

### 典型场景

- **Sigmoid**: 导数最大0.25，深层网络梯度快速消失
- **Tanh**: 导数最大1，但通常<1
- **深层网络**: 层数越多，连乘效应越强

### 症状

- 浅层参数几乎不更新
- 训练收敛极慢或停滞
- 深层网络性能不如浅层

## 梯度爆炸 (Exploding Gradient)

### 原因

当权重或激活导数连乘 > 1时，梯度指数增长。

### 症状

- 损失变为NaN或Inf
- 权重更新极大，模型发散
- 训练不稳定

## 解决方案

### 1. 激活函数选择

| 激活函数 | 导数范围 | 梯度问题 |
|---------|---------|---------|
| Sigmoid | 0 ~ 0.25 | 严重消失 |
| Tanh | 0 ~ 1 | 中等消失 |
| ReLU | 0 或 1 | 轻微消失 |
| GELU | ~0 ~ 1 | 较好 |

**推荐**: ReLU/GELU替代Sigmoid

### 2. 权重初始化

**Xavier初始化** (Sigmoid/Tanh):
```
W ~ U(-√(6/(n_in+n_out)), √(6/(n_in+n_out)))
```

**He初始化** (ReLU):
```
W ~ N(0, √(2/n_in))
```

**目标**: 使前向和反向传播时方差稳定

### 3. 归一化层

- **BatchNorm**: 每层输出归一化，稳定梯度
- **LayerNorm**: Transformer标配

### 4. 残差连接

```
y = F(x) + x
∂y/∂x = ∂F/∂x + 1  # 至少有1的梯度
```

- 梯度高速公路，直通浅层
- **应用**: ResNet、Transformer

### 5. 梯度裁剪

```python
grad = grad * min(1, threshold/||grad||)
```

- 防止梯度爆炸
- **应用**: RNN/LSTM训练

## 梯度流动诊断

### 梯度范数监控

```python
for name, param in model.named_parameters():
    print(f"{name}: grad_norm = {param.grad.norm()}")
```

### 梯度分布可视化

- 检查各层梯度统计量
- 发现消失/爆炸的层
