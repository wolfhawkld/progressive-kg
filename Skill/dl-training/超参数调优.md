# 超参数调优

> 超参数调优是寻找最优超参数组合的过程。好的超参数可以显著提升模型性能。

---

主要超参数

| 超参数 | 典型范围 | 影响 |
|-------|---------|------|
| 学习率 | 1e-5 ~ 1e-1 | 收敛速度、稳定性 |
| Batch Size | 8 ~ 512 | 梯度稳定性、训练速度 |
| 层数 | 2 ~ 100+ | 模型容量、训练难度 |
| 隐藏层维度 | 64 ~ 4096 | 模型容量、计算成本 |
| Dropout率 | 0 ~ 0.5 | 正则化强度 |
| 权重衰减 | 0 ~ 0.1 | 正则化强度 |

## 调优方法

### 网格搜索 (Grid Search)
穷举所有参数组合

### 随机搜索 (Random Search)
随机采样参数组合，效率高于网格搜索

### 贝叶斯优化 (Bayesian Optimization)
根据已有结果建立代理模型，预测更优参数
工具: Optuna、Hyperopt、Ray Tune

## 实践策略

- 学习率和batch size最重要，先调这两个
- 粗到细搜索：大范围少量样本 → 缩小范围增加样本
- 网络架构调优空间小，通常用经验值
