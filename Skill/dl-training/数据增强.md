# 数据增强

## 概念

数据增强通过对训练数据进行变换，增加数据多样性和数量，是防止过拟合、提升泛化能力的核心技术。

## 图像增强

### 基础方法
- **几何变换**: 裁剪、翻转、旋转、缩放
- **颜色变换**: 亮度、对比度、饱和度、色调

### 高级方法

**Mixup**: 混合两张图像和标签
```
x = λx₁ + (1-λ)x₂
```

**CutMix**: 裁剪区域替换

**AutoAugment/RandAugment**: 自动搜索增强策略

## 文本增强

- 同义词替换
- 随机插入/删除/交换
- 回译 (Back Translation)
- BERT-based上下文替换

## 任务相关推荐

| 任务 | 推荐增强 |
|-----|---------|
| 图像分类 | 几何+颜色+Mixup |
| 目标检测 | 几何变换（保持标注） |
| 文本分类 | 同义词替换+回译 |
