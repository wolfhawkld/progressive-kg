# 学习率调度

## 概念

学习率调度（Learning Rate Scheduling）在训练过程中动态调整学习率，平衡收敛速度和最终性能。

## 为什么需要调度

- **初期**: 大学习率快速下降
- **中期**: 适中学习率稳定优化
- **后期**: 小学习率精细调整

## 常见调度策略

### Step Decay

```
每N个epoch，学习率乘以衰减因子
η = η_0 * γ^(epoch // N)
```

- 简单有效
- 需要手动选择N和γ

### Exponential Decay

```
η = η_0 * γ^epoch
```

- 连续衰减
- 参数少

### Cosine Annealing

```
η = η_min + 0.5 * (η_max - η_min) * (1 + cos(π * t / T))
```

- 平滑衰减
- 后期学习率变化缓慢
- **应用**: Transformer、大模型标配

### Linear Warmup

```
前T_warmup步: η = η_0 * t / T_warmup
```

- 训练初期小学习率预热
- 防止早期梯度不稳定
- **应用**: Transformer标配

### Warmup + Cosine Decay

```
Warmup阶段: η线性增加
然后: Cosine衰减
```

- **应用**: BERT、GPT、LLaMA等大模型标配

### Cosine Annealing with Warm Restarts

```
周期性重置学习率，多次退火
```

- 跳出局部最优
- SGDR (Stochastic Gradient Descent with Warm Restarts)

### Polynomial Decay

```
η = η_0 * (1 - t/T)^p
```

- p: 衰减多项式次数
- 可控制衰减速度

### One-Cycle Policy

```
学习率: 小 → 大 → 小
```

- Super-convergence
- 快速训练

## 自适应调整

### ReduceLROnPlateau

```
监控验证集指标
指标停滞时降低学习率
```

- 根据实际性能调整
- patience参数控制容忍度

## 调度策略对比

| 策略 | 特点 | 适用场景 |
|-----|------|---------|
| Step Decay | 简单可控 | CNN图像分类 |
| Cosine | 平滑，后期稳定 | Transformer |
| Warmup + Cosine | 稳定训练 | 大模型标配 |
| One-Cycle | 快速收敛 | 快速训练 |
| ReduceLROnPlateau | 自适应 | 通用 |

## 实践建议

### 学习率选择

- **Adam/AdamW**: η = 1e-4 ~ 1e-3
- **SGD**: η = 0.1 ~ 0.01
- **大模型**: η = 1e-5 ~ 1e-4

### Warmup配置

- **步数**: 总训练步数的1-10%
- **大模型**: 更长的warmup（1000-10000步）

### 衰减终点

- 通常衰减到初始学习率的1/100 ~ 1/1000
- 或设置η_min = 1e-6 ~ 1e-7
