# 参数高效微调 (PEFT)

## 概念

参数高效微调只训练少量参数即可适配下游任务，大幅降低大模型微调成本。

## 主要方法

### LoRA (Low-Rank Adaptation)
```
W' = W + BA  # B: d×r, A: r×d
只训练B和A，冻结W
```
- 不增加推理延迟（可合并到原权重）
- 大语言模型微调标配

### Adapter
在Transformer层中插入小型瓶颈网络

### Prefix Tuning
在输入前添加可学习的虚拟token

### Prompt Tuning
在输入embedding层添加可学习软提示

## 方法对比

| 方法 | 参数比例 | 推理开销 | 效果 |
|-----|---------|---------|------|
| LoRA | ~0.1% | 无 | 接近全量微调 |
| Adapter | ~1% | 有 | 接近全量微调 |
| Prefix Tuning | ~0.1% | 有 | 好 |
| Prompt Tuning | ~0.01% | 有 | 中等 |
