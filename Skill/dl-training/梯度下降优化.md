# 梯度下降优化

## 概念

梯度下降是神经网络训练的核心优化方法。通过计算损失函数对参数的梯度，沿梯度反方向更新参数以最小化损失。

## 基础梯度下降

### SGD (Stochastic Gradient Descent)

```
θ = θ - η * ∇L(θ)
```

- **优点**: 简单，内存占用小
- **缺点**: 收敛慢，震荡，需要精心调学习率

### Mini-batch SGD

```
θ = θ - η * (1/m) * Σ∇L_i(θ)
```

- 使用小批量样本计算梯度
- 平衡计算效率和梯度稳定性

## 动量方法

### Momentum

```
v = β * v + ∇L(θ)
θ = θ - η * v
```

- **β**: 动量系数（通常0.9）
- **作用**: 加速收敛，抑制震荡

### Nesterov Accelerated Gradient (NAG)

```
v = β * v + ∇L(θ - η * β * v)
θ = θ - η * v
```

- 先"展望"再计算梯度
- 更好的收敛性

## 自适应优化器

### AdaGrad

```
G = G + (∇L)²
θ = θ - η * ∇L / √(G + ε)
```

- 自适应调整每个参数的学习率
- **问题**: G持续累积，学习率趋近于0

### RMSprop

```
G = β * G + (1-β) * (∇L)²
θ = θ - η * ∇L / √(G + ε)
```

- 使用指数移动平均，解决AdaGrad问题

### Adam (Adaptive Moment Estimation)

```
m = β₁ * m + (1-β₁) * ∇L     # 一阶矩（均值）
v = β₂ * v + (1-β₂) * (∇L)²  # 二阶矩（方差）
m̂ = m / (1 - β₁^t)          # 偏差修正
v̂ = v / (1 - β₂^t)
θ = θ - η * m̂ / √(v̂ + ε)
```

- **默认参数**: β₁=0.9, β₂=0.999, ε=1e-8
- **优点**: 快速收敛，对超参数不敏感
- **应用**: 深度学习标配

### AdamW

```
Adam + 解耦的权重衰减
```

- 将weight decay与梯度更新分离
- 更好的泛化性能
- **应用**: Transformer训练标配

### Adam变体

| 优化器 | 特点 |
|-------|------|
| Adamax | Adam的∞范数变体 |
| Nadam | Adam + Nesterov |
| RAdam | 修正Adam的方差 |
| Lookahead | 多个优化器的组合策略 |

## 优化器选择指南

| 场景 | 推荐优化器 |
|-----|-----------|
| 通用深度学习 | Adam |
| Transformer | AdamW |
| CNN图像分类 | SGD + Momentum |
| 小数据集 | SGD + Momentum |
| 快速原型 | Adam |
| 最佳性能 | SGD + Momentum + 调优 |
