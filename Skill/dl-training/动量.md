# 动量（Momentum）

> 优化器中的惯性机制，累积历史梯度方向，平滑震荡、加速收敛。是SGD-Momentum、Adam、Muon等优化器的核心组件。

---

## 直觉理解

**物理类比**：

```
小球滚下山坡
├─ 没有惯性 → 每一步只看当前坡度（纯SGD）
│            → 在小坑洼处停顿、震荡
└─ 有惯性 → 保持之前的运动趋势
           → 冲过局部坑洼，直奔最低点
```

**优化场景**：

| 地形特征 | 纯SGD行为 | 有动量行为 |
|----------|-----------|------------|
| 狭长山谷 | 来回震荡 | 惯性平滑，沿谷底前进 |
| 局部坑洼 | 被卡住 | 冲过去 |
| 鞍点 | 停滞 | 惯性推动前进 |
| 噪声梯度 | 被干扰 | 噪声被平滑 |

---

## 核心公式

### SGD + Momentum

```
m_t = β × m_{t-1} + g_t       # 动量累积
θ_t = θ_{t-1} - lr × m_t      # 用动量更新
```

| 参数 | 含义 | 常用值 |
|------|------|--------|
| **m_t** | 动量向量（梯度累积） | 初始为0 |
| **β** | 动量系数（惯性强度） | 0.9 |
| **g_t** | 当前梯度 | - |
| **lr** | 学习率 | - |

### Nesterov Momentum（NAG）

"先走一步，再看梯度"：

```
θ_lookahead = θ_t + β × m_t      # 先走惯性一步
g_t = ∇L(θ_lookahead)            # 在前瞻位置计算梯度
m_t = β × m_{t-1} + g_t          # 动量更新
θ_t = θ_{t-1} - lr × m_t         # 参数更新
```

**优势**：前瞻梯度更准确，减少"冲过头"。

---

## β 的选择

| β值 | 惯性强度 | 效果 |
|-----|----------|------|
| 0.0 | 无惯性 | 等于纯SGD |
| 0.5 | 弱惯性 | 响应快，但平滑效果弱 |
| **0.9** | 标准惯性 | 平衡平滑与响应（默认） |
| 0.99 | 强惯性 | 极度平滑，响应慢 |

---

## 动量 vs 纯SGD

### 鞍点问题

```
鞍点：某些方向向上，某些方向向下
纯SGD → 梯度接近0，停滞
动量 → 历史惯性推动，跨越鞍点
```

### 震荡问题

```
狭长山谷地形：
    \      /
     \    /
      \  /    ← 纯SGD来回震荡
       --
动量 → 垂直方向震荡被平滑，水平方向加速
```

### 收敛轨迹对比

```
纯SGD：曲折、震荡、慢
动量：平滑、直接、快
```

## 各优化器的动量实现

| 优化器 | 动量方式 | 特点 |
|--------|----------|------|
| **SGD-Momentum** | 简单累积 | 最基础 |
| **Nesterov** | 前瞻梯度 | 更准确 |
| **Adam** | 一阶矩估计 + 二阶矩自适应 | 动量+自适应学习率 |
| **AdaBound** | Adam + 动态边界 | 动量自适应范围限制 |
| **Muon** | 动量矩阵 → Newton-Schulz正交化 | 矩阵级动量处理 |

---

## Adam 的动量（一阶矩）

Adam 中的动量称为"一阶矩估计"：

```
m_t = β₁ × m_{t-1} + (1-β₁) × g_t    # 动量（一阶矩）
v_t = β₂ × v_{t-1} + (1-β₂) × g_t²   # 二阶矩（自适应学习率）
m̂_t = m_t / (1-β₁^t)                # 偏差修正
v̂_t = v_t / (1-β₂^t)
θ_t = θ_{t-1} - lr × m̂_t / √v̂_t
```

- β₁ 通常 = 0.9（动量系数）
- β₂ 通常 = 0.999（自适应系数）

---

## Muon 的动量

Muon 对动量进行**正交化**处理：

```
M_t = β × M_{t-1} + G_t              # 动量矩阵
O_t = Newton-Schulz(M_t)             # 正交化
θ_t = θ_{t-1} - lr × O_t             # 用正交化动量更新
```

详见 [[Muon优化器]]。

## Python 实现

### SGD + Momentum

```python
class SGD_Momentum:
    def __init__(self, lr=0.01, momentum=0.9):
        self.lr = lr
        self.momentum = momentum
        self.v = None  # 动量
    
    def update(self, params, grads):
        if self.v is None:
            self.v = {k: 0 for k in params}
        
        for k in params:
            self.v[k] = self.momentum * self.v[k] + grads[k]
            params[k] -= self.lr * self.v[k]
```

### Nesterov Momentum

```python
class Nesterov_Momentum:
    def __init__(self, lr=0.01, momentum=0.9):
        self.lr = lr
        self.momentum = momentum
        self.v = None
    
    def update(self, params, grads_func):
        if self.v is None:
            self.v = {k: 0 for k in params}
        
        # 前瞻位置
        lookahead = {k: params[k] + self.momentum * self.v[k] for k in params}
        
        # 在前瞻位置计算梯度
        grads = grads_func(lookahead)
        
        # 更新动量和参数
        for k in params:
            self.v[k] = self.momentum * self.v[k] + grads[k]
            params[k] -= self.lr * self.v[k]
```

## 动量的物理意义

苏剑林在《从动力学角度看优化算法》中指出：

```
动量优化 ≈ 在参数空间中的"带惯性的运动"
```

| 物理概念 | 优化对应 |
|----------|----------|
| 位置 | 参数 θ |
| 速度 | 动量 m |
| 力 | 梯度 -∇L |
| 质量 | 1/β（惯性大小） |
| 势能 | 损失函数 L(θ) |

## 实际使用建议

| 场景 | 推荐设置 |
|------|----------|
| **通用场景** | β=0.9 |
| **噪声数据** | β=0.95（更强平滑） |
| **快速响应** | β=0.7-0.85 |
| **配合学习率** | 动量强 → 学习率可适当增大 |

## 记忆要点

| 要点 | 内容 |
|------|------|
| **公式** | m_t = βm + grad，用m更新而非grad |
| **β=0.9** | 标准惯性，保留90%历史趋势 |
| **作用** | 平滑震荡、冲过鞍点、加速收敛 |
| **Nesterov** | 先走惯性一步再算梯度，更准确 |
| **Adam一阶矩** | 带偏差修正的动量 |
